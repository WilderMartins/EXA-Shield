# Como Coletar e Enviar os Logs de Instalação

Se a instalação travou ou apresentou algum erro, o arquivo de log é essencial para que possamos entender o que aconteceu. Siga os passos abaixo para nos ajudar a diagnosticar o problema.

## Passo 1: Localize o Arquivo de Log

O assistente de configuração salva um registro detalhado de todas as suas ações em um arquivo chamado `setup-gcp.log`. Este arquivo está localizado na mesma pasta onde você executou o comando de instalação.

## Passo 2: Copie o Conteúdo do Log

Você pode abrir o arquivo `setup-gcp.log` com qualquer editor de texto ou usar o comando abaixo no seu terminal para exibir o conteúdo:

```bash
cat setup-gcp.log
```

## Passo 3: Compartilhe o Log

Copie **todo o conteúdo** exibido pelo comando `cat` e cole-o na nossa conversa. Isso nos dará as informações necessárias para identificar a causa do problema e te ajudar a concluir a instalação.
